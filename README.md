Информационный сайт на тему "Туристические эксурсии"
Создан для ознакомления с проводимыми экскурсиями в туристических городах

Реализованная функциональность:
1. Обзор туристических экскурсий трёх городов: Сочи, Пицунды и Геленджика
2. Просмотр картинок популярных мест и краткий экскурс в историю

Особенность проекта в следующем:
1. Телефонный звонок для уточнения наличия туристических экскурсий и брони

Основной стек технологий:
1. HTML, CSS, JavaScript
2. Bottle
3. Git

Среда запуска:
1. Развёртывание происходит в среде Visual Studio
2. Требуется Python версии не ниже 3.10

Установка Python 3.10
1. Передите на сайт Python по ссылке https://www.python.org/ftp/python/3.10.1/python-3.10.1-amd64.exe
2. Поставьте галочку напротив "ADD Python 3.10 to PATH"
3. Готово
